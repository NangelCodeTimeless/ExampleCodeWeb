Si desea que se desplace suavemente hacia la parte superior una vez que se complete la animación, una cosa que podría hacer es modificar un poco su animación, para que en lugar de terminar en la parte inferior, termine en la parte superior (ese es el original estado para que no haya ningún "salto" cuando se reinicie la animación).

Puedes hacer esto:

  • Desplazarse hacia abajo usando el 95% del tiempo de animación.
  • Usando el 5% restante para desplazarse hacia arriba.

Los cambios en el código serían realmente simples y no requerirían JavaScript:

@keyframes scroll {
    95% { top: -100%; }
    100% { top:0; }
}

Esta demostración muestra el resultado:

.c{
  position:relative;
  background:red;
  max-height:200px;
  float:left;
  width:300px;
  height:300px;
  overflow:hidden;
  overflow-y:auto;
}
.card-home{
  position:absolute;
  margin:20px;
  top:0;
  animation: scroll 10s linear 1s infinite;
}
span {
  min-width:300px;
  min-height:40px;
  display:block;
  color:white;
  margin:5px;
  background:blue;
}
@keyframes scroll {
  95% { top: -100%; }
  100% { top:0; }
}
